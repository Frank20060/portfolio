## Qué hice

- Implementación de un panel administrativo (CMS) para gestionar el contenido educativo de forma dinámica.
- Desarrollo de un sistema de autenticación de usuarios y registro de progreso en ejercicios prácticos.
- Integración de interfaz CRUD con pestañas interactivas para preguntas y ejercicios técnicos de Shopify.
- Creación de un sistema de procesamiento y renderizado de archivos Markdown para documentar guías de desarrollo.

## Resultado

Una plataforma educativa escalable y autogestionable que mejora la experiencia de aprendizaje de desarrollo de temas y apps de Shopify.

## Stack

- Next.js / React
- Node.js / Express
- MongoDB
- Tailwind CSS
